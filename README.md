# CopyShell

## 项目简介

CopyShell 是一个基于 Windows 原生 `robocopy.exe` 的资源管理器增强复制工具，支持 Windows 10 1809+ 和 Windows 11。

它不会替换系统自带的复制粘贴功能，而是在资源管理器右键菜单中增加：

- **高级复制到…**
- **高级移动到…**
- **同步到…**

CopyShell 使用 C#、WinUI 3 和原生 Windows Shell Extension 构建，面向大文件、批量文件和可靠目录同步场景。

> 项目目前处于开发准备阶段，尚未发布可运行版本。

## 使用方式

发布版本安装完成后：

1. 在 Windows 资源管理器中选择文件或文件夹。
2. 打开右键菜单并选择 `CopyShell`。
3. 选择 `高级复制到…`、`高级移动到…` 或 `同步到…`。
4. 选择目标文件夹，确认任务选项后开始执行。

Windows 11 用户需要从“显示更多选项”中打开经典右键菜单。

> `同步到…` 会镜像源文件夹，并可能删除目标中多余的文件，执行前请仔细确认目标路径。
