# CopyShell

## 项目简介

CopyShell 是一个基于 Windows 原生 `robocopy.exe` 的资源管理器增强复制工具，支持 Windows 10 1809+ 和 Windows 11。

它不会替换系统自带的复制粘贴功能，而是在资源管理器右键菜单中增加：

- **高级复制到…**
- **高级移动到…**
- **同步到…**

CopyShell 使用 C#、WinUI 3 和原生 Windows Shell Extension 构建，面向大文件、批量文件和可靠目录同步场景。它支持任务队列、暂停恢复、执行历史、精确进度和失败重试。

## 使用方式

安装发布页中的 `CopyShell.appinstaller` 后：

1. 在 Windows 资源管理器中选择文件或文件夹。
2. 右键选择 `CopyShell`；Windows 11 会显示在第一层菜单。
3. 选择 `高级复制到…`、`高级移动到…` 或 `同步到…`。
4. 选择目标文件夹，确认选项后把任务加入队列。
5. 在任务列表中查看进度，或暂停、恢复、取消和重试任务。

Portable 版本运行 `install.ps1` 完成当前用户安装；它使用经典右键菜单，Windows 11 中位于“显示更多选项”。

> `同步到…` 会镜像源文件夹，并可能删除目标中多余的文件，执行前请仔细确认目标路径。
